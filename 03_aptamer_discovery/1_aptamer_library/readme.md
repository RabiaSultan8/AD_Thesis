# Phase II-1: RNA Aptamer Library Generation & Filtering

This module contains the codebase for the de novo generation of the RNA aptamer library and the subsequent sequential filtering pipeline.

## Execution Pipeline
The pipeline is divided into two primary scripts. 

### 1. Library Generation & 1D/2D Filtering
* **`aptamers_generation_filter.py`**: A master automation script executing the following sequence:
    * **Generation**: Creates a 5,000-sequence library across four structurally motivated, G-rich classes (G4-Core, UGGG-Anchor, Distributed-G, Mixed Scaffold) using a fixed random seed (`1122`) for exact reproducibility.
    * **RNAfold**: Predicts secondary structures and calculates minimum free energy (MFE) and ensemble diversity.
    * **Biophysical Filter**: Applies six stringent thresholds (e.g., MFE $\le$ -8.0 kcal/mol, loop size 10-14 nt).
    * **Sequence Clustering**: Uses CD-HIT-EST to remove $>80\%$ sequence redundancy.
    * **Structural Clustering**: Computes an all-vs-all RNApdist distance matrix and performs hierarchical average-linkage clustering (cutoff = 1.0) to isolate unique thermodynamic ensemble profiles.
    * **Reporting**: Generates the final 257-aptamer sequence library and visualizes the attrition metrics.

*(Note: The output from this stage was submitted to the trRosettaRNA web server for 3D structure prediction. The resulting PDB models are required for the next step.)*

### 2. 3D Quality Control
* **`filter_3d_aptamers.py`**: Evaluates the 3D models generated by trRosettaRNA. 
    * Checks for atomic clashes (distance < 0.5 Å).
    * Calculates total solvent-accessible surface area via FreeSASA (requires > 500 Å²).
    * Removes 3D structural redundancy using US-align (TM-score cutoff = 0.85).

## Output
The final output of this module is a curated library of 244 clash-free, SASA-sufficient, and structurally non-redundant 3D aptamer models ready for blind docking in the `2_HDOCK` module.
